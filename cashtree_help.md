📕 사용법 안내 📕


📌 검색어를 입력하면 문제에서 검색어와 일치하는 문제를 알려줍니다. 예) 고덕치과

✅ 일치하는 문제가 여러개 이면 원하는 문제의 번호를 선택합니다.

  ▶ 일치하는 문제와 답을 한번에 출력하려면 /nonlist 를 입력하세요.

  ▶ 자세한 설정은 "📌 검색어 출력 모드 설정" 을 참고하세요.

✅ 검색어는 콤마(,) 를 구분자로 AND 검색 됩니다. 예) 진주,행복

✅ 초성검색도 가능합니다. 예) ㄱㅅㄱㅈㅈ


📌 정답갯수 설정

✅ 저장된 정답을 알려주는 기본 갯수는 {maxAnswerCnt} 개 입니다.

✅ 정답알림 갯수를 변경하려면 /answer 후 갯수 또는 :갯수 를 입력합니다. 예) /answer 5 또는 :5

✅ 특정문제만 정답갯수를 변경해서 볼 수 있습니다. 예) 블랙박스:7


📌 검색어 출력 모드 설정

✅ 검색어 출력 기본은 선택한 문제의 답을 출력합니다.

✅ 문제와 답을 한번에 출력 하려면 /nonlist 를 입력하세요.

✅ /nonlist 는 toggle 기능으로 명령어를 입력할 때마다 설정이 바뀝니다.


📌 이미지 출력 모드 설정

✅ 검색된 문제의 이미지 출력 모드 기본은 작게 출력합니다.

✅ 이미지를 크게 하려면 /image 를 입력하세요.

✅ /image 는 toggle 기능으로 명령어를 입력할 때마다 설정이 바뀝니다.


📌 설정정보 조회

✅ /status 또는 /s 를 입력하세요.
